# Transfer Learning para Clasificación de Flores con EfficientNetB0

## Descripción

Clasificación de 104 tipos de flores utilizando el modelo base preentrenado EfficientNetB0 y un conjunto de datos de Kaggle. El rendimiento del modelo se evalúa mediante la métrica F1 Score.

## Documentos

* Código de Python usado para el proyecto en flowers.py
* Modelo creado a traves del código en flowers.onnx
* Página web para el reconocimiento de las flores con cámara en index.html
* Imágenes para la página web

## Requisitos

Para ejecutar el proyecto, necesitarás:

* Python 3.10.19
* Las dependencias de Python listadas en 'requirements.txt'
